# Narola Infotech — Company Profile Dashboard

A research dashboard for **Narola Infotech Private Limited** (CIN U72900GJ2013PTC074532), aggregating public data across six modules:

| Module | Description |
|--------|-------------|
| **Company Profile** | Overview, leadership, Trustpilot reviews, office locations, registration details |
| **AmbitionBox Analysis** | Salaries, work culture reviews, ratings, benefits, notice period data |
| **Freelancer Reviews** | 949 client reviews with charts, repeat-client tracking, revenue trends & filters |
| **Mobile App Development** | App teardowns — QuadMaps & VoetstAPPen (Play Store data, user reviews) |
| **Org Chart** | Company hierarchy, departments, management structure |
| **People Directory** | 10 key employee profiles — roles, experience, education, contact info |

Built with vanilla HTML/CSS and Font Awesome icons. Hosted on GitHub Pages.
